一、必接引脚 (单音正弦输出 = Mode 1 最小系统)
--------------------------------------------------------------
    STM32F103VE     方向        AD9910 模块丝印     芯片功能
    -----------     -----       ---------------     -----------------------
    PA5             ->          SCLK                 SPI 时钟   (SPI1 SCK, 9 MHz)
    PA7             ->          SDIO                SPI 数据   (SPI1 MOSI, 只写)
    PA4             ->          CSB                 片选       (低有效, 软控)
    PC3             ->          RST                 MASTER_RESET (高电平复位)
    PC4             ->          UPD                IO_UPDATE   (上升沿刷寄存器)
    PB12            ->          PF0            (PF0,PF2,PF2的功能描述)Mode 4 跳频用. 单音模式下必须为低电平,否则输出可能偶发丢失.
    PB13            ->          PF1            推荐做法二选一:
                                                A) 三根线连到 STM32, 让驱动拉低
                                                B) 模块侧直接短接到 GND (更省 GPIO)
                                                悬空 = 错误配置, 会不稳定.
    PB14            ->          PF2            
    GND             <->         GND                 共地 (必须!)

    电源: 外部 DC 5V (0.5A 以上)  ->  模块 "DC-5V" USB 供电座，连接到电源的CH3USB接口即可，成功供电的时候AD9910模块会有小蓝灯亮起来
          !! AD9910 模块不能从 STM32 板取 3.3V, 它自带 5V->3.3V/1.8V 稳压器 !!

三、完全不接的引脚 (悬空 OK)
--------------------------------------------------------------
    SDO       SDO引脚是无排针的引脚，有些资料会提到这个引脚，实际上我们是无法控制的
    PWR      EXT_PWR_DWN, 掉电控制脚, 模块上无此引脚排针，无法控制
    SYC       SYNC 相关, 多片同步用
    PDCLK     并口数据时钟, 并口模式用
    TEN       TXENABLE, 并口模式用
    RSO       IO_RESET, 通信复位用
    PLL       无芯片对应排针
    DRO       DROVER, 是芯片输出脚
    D0..D15   16 位并口数据总线, 并口模式用
    F0, F1    并口目的选择, 并口模式用

    -> 全部悬空, 本工程不需要
AD9910是一个有90多个引脚的芯片，而AD9910模块上的排针只引出了一部分引脚，有些资料虽然提到了一些引脚，但是实际上我们无法控制，比如上面的SDO,PLL,PWR。
并口模式是这样，AD9910可以通过SPI串行输入，也可以直接并行输入，都可以输出一样的波形。一般有FPGA用并行，无的话用串行，所以并行相关引脚我们一般都不用，悬空即可。

四、示波器接线
--------------------------------------------------------------
    AD9910 SMA 输出       示波器
    ---------------       ---------------------------
    OUT+ (J3)             CH1     
    OUT- (J4)             也可以接 CH2 做差分，观察CH1-CH2波形，也可以不管。这个信号是OUT+的反相，不差分没事情。

    AD9910芯片输出是电流，但是AD9910模块内置了一些电路将电流转化为电压，将模块的SMA接一个SMA转鳄鱼夹线(直接和老师说要这个线)，再连接示波器就可以用，连接排针则可接入实际电路。

五、幅度校准 (已实测)
--------------------------------------------------------------
    include/module/dds.h 里 DDS_FULL_SCALE_V = 1.0f
    实测: dds_tone_sine(f, 1.0f, 0) 时示波器测到差分峰值约 1 V
    -> 当前 1.0f 已经匹配本板实际满量程, 不需要重新校准.
    -> 若换其他 AD9910 模块 (balun / Rset 不同), 需重新实测并修改此宏.
